# Sprint 02 회고

> 이번 스프린트도 열심히 했나?

나는 sprint-02 동안 정말 열심히 한 걸까? 

스프린트가 끝나고 결과물과 진행률을 보니 열심히 했다고 한 것에 비해 많이 아쉽다.

왜 이번에도 아쉬울까?

1. **규칙적인 생활**

    가장 큰 이유라고 생각 된다. 하루 밤새면 다음 날에 아무 것도 못하게 되는 현상이 너무 빈번하게 발생하는 것 같아 다음 스프린트는 규칙적인 생활에서 시작해야 겠다고 생각했다.
2. **서두르지 않기**

    뭔가 조급해지는 느낌을 받지만, 그냥 하자. 매일 하면 달라 질 테니까 그냥 아무 생각 없이 하자.

스프린트 3에서는 위의 2가지를 반드시 지키면서 진행해봐야 겠다.


> 시간 배분

오늘 프로젝트 기획서를 제출했고, 이 후 본격적인 프로젝트가 시작될려고 한다.

프로젝트, 학습, 미니 프로젝트 시간 배분을 정말 잘해야 한다. 전 날에 어디에 몇 시간을 소비할 지 미리미리 정해야 하겠다는 생각이 들었다. 

내가 하루에 사용할 수 있는 총 시간은? **10시간**

정말 소중한 시간이다. 매일 저녁 이 10시간을 어떻게 사용할 지, 정하지 않는다면 시간에 쫒기는 하루가 될 같다.

이는 내가 제일 많이 보는 바탕화면 캘린더에 작성해보겠다.

> 저번 스프린트보다 성장한 것

일단 그냥 하자라는 마인드가 더 몸에 배인 것 같다. 저녁에도 그냥 스터디 카페가서 한 시간이라도 공부하자는 생각이 들고, 몸이 바로 움직이는 것 같아서 이 점은 확실히 성장한 것 같다고 느꼈다.

자바라는 프로그래밍 언어에 대해서 한층 더 성장한 것 같다.

개발자라는 직무에서 나의 위치를 더 정확히 알았다. `밑바닥..ㅎ` 하지만! 그래서 어떻게 해야할 지 더 명확해진 것 같다. 그냥 하보자
